## 📑　사내 결재 시스템
사내에서 오고 가는 문서를 전자로 결재하기 위한 웹 애플리케이션입니다.  
업무의 실수를 줄이고, 출장과 미팅 중에도 업무를 처리할 수 있어 효율을 높일 수 있습니다.

## 💡　주요 기능
- 회원가입
  - 등록된 정보를 기반으로 DB에 유저 등록
  - Open API 활용한 자택 주소 확인 및 유저 이메일 인증
<br>

- 로그인
  - 유효성 검사를 통한 유저 확인
  - 세션을 활용한 유저 데이터 저장 및 추출
<br>

- 결재 시스템
  - 결재 글 임시 저장 및 작성
  - 결재 글 조회 및 검색
  - 직급에 따른 결재 승인 및 반려
  - 결재건과 연관된 경우만 결재글 조회
  - 결재 이력 확인
<br>

- 대리 결재 시스템
  - 부장 및 과장 직급의 경우 하위 직급에게 대리로 권한 위임
  - 24시간 동안 대리자 역할 수행
<br>
